# AI Game Collection

## 黑洞吞噬世界

一款有趣的HTML5游戏，玩家控制一个黑洞，通过吞噬各种物体来增长体积。

### 游戏特色

- **无限地图**: 在5000x5000像素的大地图中自由探索
- **多样物体**: 可以吞噬草、石头、树木、汽车、房屋、动物、人类、外星人和飞船
- **物理效果**: 物体有重力效应，越靠近黑洞移动越慢
- **动态场景**: 动物和人类会试图逃离黑洞
- **音效系统**: 包含背景音乐和吞噬音效
- **视觉效果**: 黑洞有旋转动画和辐射效果

### 操作方法

- 方向键或WASD控制黑洞移动
- 吞噬比自己小的物体来增长体积
- 避免与比自己大的物体接触
