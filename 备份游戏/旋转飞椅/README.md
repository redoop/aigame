# 旋转飞椅

一款使用 Three.js 制作的 3D 旋转飞椅游戏。

## 游戏特色

- **3D 场景**: 使用 Three.js 渲染真实的 3D 旋转飞椅
- **物理效果**: 飞椅随速度产生离心力摆动
- **交互控制**: 可控制启动/停止、加速/减速、升高/降低
- **视角控制**: 鼠标拖动旋转视角，滚轮缩放
- **光影效果**: 真实的光照和阴影系统

## 操作方法

- **启动/停止**: 点击"启动"按钮开始或停止旋转
- **加速/减速**: 控制飞椅旋转速度
- **升高/降低**: 调整飞椅高度（5-20米）
- **鼠标拖动**: 旋转观察视角
- **鼠标滚轮**: 缩放视角距离

## 游戏元素

- 🎡 中心支柱
- 💺 8个彩色座椅
- ⛓️ 金属链条
- 🌅 动态光影
- 🌿 草地场景

## 技术实现

- Three.js 3D 渲染引擎
- 物理模拟（离心力、摆动）
- 实时阴影系统
- 平滑动画过渡
- 响应式设计

## 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。
