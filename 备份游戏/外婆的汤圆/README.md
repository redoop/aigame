# 外婆的汤圆

一款温馨的3D互动游戏，和外婆一起包金华咸味汤圆。

## 游戏特色

- **Three.js 3D场景**: 立体的外婆、玩家和汤圆模型
- **中文语音配音**: 使用Web Speech API实现实时语音
- **背景音乐**: 温馨的旋律循环播放
- **三个阶段**: 观看→一起包→一起吃

## 操作方法

1. 点击"开始"按钮
2. 观看外婆包汤圆
3. 点击屏幕一起包汤圆（需包7个）
4. 自动进入吃汤圆环节

## 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。

## 技术栈

- HTML5 + CSS3
- Three.js (3D渲染)
- Web Audio API (音效)
- Web Speech API (语音合成)
