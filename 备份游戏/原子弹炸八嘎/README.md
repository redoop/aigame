# 原子弹炸八嘎 💣

一款基于 HTML5 Canvas 的战略轰炸游戏，驾驶轰炸机投放原子弹消灭敌人！

## 🎮 游戏特色

### 武器系统
- **胖子原子弹**：经典原子弹，标准威力和范围
- **沙皇氢弹**：史上最强氢弹，1.5倍爆炸范围，2倍弹坑深度

### 敌人类型
- **掩体八嘎**：躲在混凝土掩体中，掩体有3点血量
- **跑动八嘎**：会从掩体中跑出来，击杀得分更高（200分）
- **地道八嘎**：藏在地下，需要多次轰炸击穿地面才能消灭（300分）

### 特殊目标
- **核电站**：炸毁后引发连锁爆炸，消灭所有地面敌人（每个500分）
- **核电站重建**：被炸毁10秒后八嘎会重建核电站

### 游戏机制
- **掩体血量系统**：掩体有3点血量，需要多次轰炸才能摧毁
- **弹坑系统**：爆炸会在地面留下弹坑，累积深度可炸穿地面
- **蘑菇云特效**：真实还原原子弹爆炸效果（闪光→火球→蘑菇云）
- **音效系统**：发射和爆炸音效

## 🎯 操作方式

- **鼠标点击**：控制轰炸机飞往目标位置
- **空格键**：按住连续投放原子弹（地毯式轰炸）
- **1键**：切换到胖子原子弹
- **2键**：切换到沙皇氢弹

## 🏆 得分系统

| 目标 | 分数 |
|------|------|
| 掩体内的八嘎 | 100分 |
| 跑出掩体的八嘎 | 200分 |
| 地道内的八嘎 | 300分 |
| 核电站连锁消灭 | 500分/个 |

## 📝 游戏策略

1. **集中火力**：对同一掩体连续轰炸以突破血量
2. **等待时机**：八嘎跑出来时击杀可获得双倍分数
3. **地毯轰炸**：对地道区域持续轰炸以累积弹坑深度
4. **核电站战术**：等敌人聚集后炸毁核电站可快速清场
5. **沙皇优势**：使用沙皇氢弹可更快击穿地面和摧毁掩体
