### 冲就完事了模拟器
一款3D清洁模拟游戏，使用Three.js开发。

**游戏特色：**
- 🎮 9个不同场景关卡（脏车、房屋、游乐场、机器人、厨房、教室、肯德基、大厦、蜜雪冰城）
- 🔫 5种水枪武器（标准、散射、高压、火箭筒、原子弹）
- 📷 3种视角模式（第一人称、第三人称、自由视角）
- 💥 火箭筒自动追踪脏污
- ☢️ 原子弹带蘑菇云特效
- 🎯 自由视角支持WASD移动和鼠标滚轮缩放

**操作说明：**
- WASD - 移动
- 鼠标 - 控制视角
- 左键 - 喷水
- V键 - 切换视角
- G键 - 切换水枪
- Q/E - 自由视角上下移动
- +/- - 自由视角缩放
